# UniMex Smart Contract

UniMex 交易所智能合约 - wstUNX Token

## 项目简介

UniMex 是一个去中心化交易所项目。本项目包含 UniMex 的核心智能合约代码。

## 智能合约

### wstUNX Token

`wstUNX` 是一个基于 ERC20 标准的代币合约，代表 Wrapped Staked UNX。

**合约特性：**
- 总供应量：1,000,000,000 wstUNX
- 代币名称：Wrapped Staked UNX
- 代币符号：wstUNX
- 支持销毁功能

## 技术栈

- **Solidity**: ^0.8.0
- **Hardhat**: 开发框架
- **OpenZeppelin**: 安全合约库
